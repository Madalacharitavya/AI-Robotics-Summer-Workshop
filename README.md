# AI & Robotics Summer Workshop

A full-stack workshop registration platform built using React, Node.js, Express, and MongoDB.

## Features

* Student Registration
* Form Validation
* Duplicate Email & Phone Prevention
* Admin Dashboard
* Search Students
* Edit Registration
* Delete Registration
* CSV Export
* Registration Analytics
* Charts Dashboard
* MongoDB Integration
* Timestamp Tracking

## Tech Stack

Frontend:

* React
* CSS
* Axios
* Chart.js

Backend:

* Node.js
* Express.js

Database:

* MongoDB Atlas

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
node server.js
```
